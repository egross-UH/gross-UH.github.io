# Elizabeth Gross faculty website

A ready-to-deploy static website.

## Deploy on GitHub Pages
1. Create a new GitHub repository, for example `egross-website`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/root`, then save.
6. GitHub will provide a public website URL after the first build.

## Edit
- Text content is in the `.html` files.
- Visual styling is in `assets/css/style.css`.
- Images are in `assets/images/`.
- CV PDF is in `assets/docs/Elizabeth-Gross-CV.pdf`.
